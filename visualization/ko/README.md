
# Install Yeoman
`npm install -g yo`

# Install knockout for yeoman
`npm install -g generator-ko`
`yo ko:component autocomplete`

# Add on another module via bower
`bower install wikimetrics-chart`

```
./node_modules/gulp

```

`python -m SimpleHTTPServer 8000`
